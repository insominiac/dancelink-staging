🧪 How to Test Dynamic Translation

Step-by-Step Testing Guide:

  1. Navigate to: http://localhost:3000
  2. Locate Language Switcher: Top-right corner with flags 🇺🇸 🇪🇸 🇰🇷 🇻🇳
  3. Click Spanish Flag (🇪🇸): Watch page content change instantly
  4. Navigate Between Pages:
    • Classes (/classes) → "Clases de Danza"
    • Events (/events) → Button text changes to Spanish
    • Contact (/contact) → "Contáctanos"
    • Instructors (/instructors) → "Nuestros Instructores"
  5. Test Persistence: Refresh page → Language should remain Spanish
  6. Test Common Elements:
    • Loading states → "Cargando..."
    • Buttons → "Ver detalles", "Reservar"
    • Status messages → "Agotado", "cupos disponibles"
